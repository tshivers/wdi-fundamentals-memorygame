Instructions

Concentration, also known as Match Match, Memory, Pelmanism, Shinkei-suijaku, Pexeso or simply Pairs, is a card game in which all of the cards are laid face down on a surface and two cards are fliped face up over each turn. The object of the game is to turn over pairs of matching cards.

About Memory Game

Memory Game was designed to help users develop and enhance their memory. Memory Game is a fun and interactive match game that increases each players ability to match components of the game while increasing memory and concentration. Try Memory Game to see how much fun and excitement memory matching can be. Invite other players and challenge them to see who can rack up the most points!
